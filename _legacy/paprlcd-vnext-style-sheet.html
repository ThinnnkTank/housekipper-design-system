Foundations

Color tokens

Ink, border, and signal are the core visual primitives. Swift should bind to semantic roles, not raw hex values.

Ink
All ink values share the same base hue (#1C1C1A). Opacity variants create hierarchy: primary text at full, strong text at 82%, secondary labels at 68%, muted meta at 50%, faint text and hairlines at 22%, hover fills at 11%.

Status / Signal
Orange is the only color deviation from monochrome. Use it exclusively for attention, notifications, and urgent states. Do not use signal color for ordinary interactive affordances.

# SemanticToken — Text

Ink-scale text colors. Light/dark via Asset Catalog.

| Token | Maps to | Use |
|---|---|---|
| `TextToken.primary`   | `Colors/ink`    | Headings, body text |
| `TextToken.strong`    | `Colors/ink80`  | Strong emphasis |
| `TextToken.secondary` | `Colors/ink60`  | Secondary text, helper |
| `TextToken.muted`     | `Colors/ink40`  | Captions, healthy status |
| `TextToken.faint`     | `Colors/ink20`  | Disabled-ish, hint |
| `TextToken.onSignal`  | `Colors/paper`  | Text on signal fill |
| `TextToken.onAction`  | `Colors/paper`  | Text on filled action |

**Status:** locked.
**Source:** `SemanticTokens/TextTokens.swift`.
